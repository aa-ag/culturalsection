# Cultural Section

### what is this?

A responsive single-page app to compile events organized by consulates in different cities.

### environment

- server: `source csenv/bin/activate`
- client: `npm run serve`

### building blocks

**infrastructure**
- `aws`
- `postgres`

**server**
- `Python`
- `Flask`
- `Flask-Login`
- `Flask-SQLAlchemy`
- `Flask-admin`

**client**
- `Vue.js`
- `vue-simple-calendar`
- `Bootstrap`
- `axios`
